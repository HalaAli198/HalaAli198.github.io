Current Research Projects

MEM-SBOM: Memory-based Runtime Software Bill of Materials Extraction for Python Applications

Developing Volatility 3 plugins to generate accurate runtime SBOMs, analyze in-memory object structures, construct dependency graphs, detect hidden or dynamically injected components, and analyze the reachability of vulnerable third-party code across application modules.

Memory Forensics of Golang Malware

Developing Volatility 3 plugins to reconstruct runtime artifacts of Go-based malware, including function metadata, goroutines, channels, strings, interfaces, and heap objects.

Runtime Detection of Malicious ML Models

Building a runtime detection system for malicious ML models that combines framework-level instrumentation with eBPF-based syscall tracing to detect attacks during model loading and execution.
