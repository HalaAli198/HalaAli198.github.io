01 About

I am a PhD candidate in Computer Science at Virginia Commonwealth University. My research focuses on memory forensics, malware analysis, software supply chain security, runtime SBOM generation, and AI security through eBPF-based monitoring.

My work develops memory- and runtime-based techniques for recovering software behavior after execution, including Volatility 3 plugins for Python and Go runtime analysis, runtime SBOM generation from Python memory, and automated memory forensics techniques for analyzing Go malware. I also investigate security risks in machine learning model supply chains, applying kernel-level tracing to detect malicious behavior at inference time.

05 Current Research Projects

Memory Forensics of Go Malware

Developing Volatility 3 plugins to reconstruct runtime artifacts of Go-based malware, including function metadata, goroutines, channels, strings, interfaces, and heap objects. Evaluated against real-world malware families including BRICKSTORM, Obscura, and Pantegana.

MEM-SBOM: Memory-based Runtime SBOM Generation

Developing memory forensic techniques to generate runtime SBOMs for Python applications, recover dependency graphs, identify dynamically injected components, and analyze vulnerable-code reachability from memory.

AI Security & ML Model Supply Chain

Investigating security risks in machine learning model distribution through eBPF-based kernel-level tracing and behavioral analysis, detecting malicious activity during model loading and inference across major ML frameworks.

3D Printing Security

Designing techniques for detecting malicious G-code manipulations and investigating illegal 3D printing activities using machine learning and memory forensics.
